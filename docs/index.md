# OmniProof Documentation

**Causal-Multimodal Engine for Creative Performance Attribution**

OmniProof embeds creatives (video, image, PDF, audio) via Gemini Embedding 2, extracts structured metadata, estimates causal effects with Double Machine Learning, and checks brand compliance via RAG — turning creative performance data into actionable, statistically rigorous proof.
